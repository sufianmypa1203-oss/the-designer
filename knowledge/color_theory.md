# Color Theory — Designer Knowledge

## The 60-30-10 Rule
- **60% Dominant**: Background/field color — sets the tone
- **30% Secondary**: Text, cards, supporting elements
- **10% Accent**: CTAs, highlights, focal elements
- This ratio creates visual balance and prevents chaos
- Breaking this ratio = amateur hour

## Color Temperature
| Temperature | Colors | Emotions | Use When |
|-------------|--------|----------|----------|
| **Warm** | Reds, oranges, yellows | Urgency, energy, passion | Pain points, CTAs, warnings |
| **Cool** | Blues, purples, greens | Trust, calm, authority | Data, professional, solutions |
| **Neutral** | Grays, whites, blacks | Sophistication, timeless | Backgrounds, text, structure |

## WCAG Contrast Requirements
- **AA Normal Text**: ≥4.5:1 contrast ratio
- **AA Large Text** (≥24px or ≥18.66px bold): ≥3:1
- **AAA Normal Text**: ≥7:1
- **AAA Large Text**: ≥4.5:1
- Always calculate: `(L1 + 0.05) / (L2 + 0.05)` where L1 is lighter

## Color Psychology
| Color | Primary Association | Secondary | Avoid For |
|-------|-------------------|-----------|-----------|
| Red | Urgency, danger, passion | Sale, warning | Trust-building |
| Blue | Trust, authority, calm | Corporate, tech | Urgency/warning |
| Green | Growth, health, money | Eco, go/success | Danger/warning |
| Yellow | Attention, optimism | Caution, energy | Professional/serious |
| Purple | Premium, creative | Luxury, wisdom | Budget/value |
| Orange | Friendly, energetic | CTA, fun | Serious/corporate |
| Black | Power, sophistication | Luxury, tech | Friendly/casual |
| White | Clean, minimal | Medical, peace | Dark themes |

## Practical Rules
1. Never use pure black (#000) on pure white (#FFF) — too harsh. Use #1A1A1A on #F5F5F5
2. Saturated colors fight for attention — only ONE per scene
3. Desaturated colors recede — use for backgrounds
4. Color alone cannot communicate — always pair with another indicator
5. Test in grayscale: if hierarchy breaks, your values are wrong
