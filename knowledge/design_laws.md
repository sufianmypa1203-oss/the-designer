# Design Laws — Designer Knowledge

## Hick's Law
- Decision time increases with the number of choices
- Maximum 4 information items per scene (pipeline standard)
- If >4 items → split into multiple scenes

## Fitts's Law
- Time to reach a target is proportional to distance/size ratio
- **Implication**: CTA elements must be large and easily targetable
- Touch targets: minimum 44×44px (132×132 on canvas)

## Miller's Law
- Working memory holds 7±2 items
- We enforce ≤4 for video (viewers are passive, not actively memorizing)

## Von Restorff Effect (Isolation Effect)
- A unique item among similar items is remembered
- Use this for focal point isolation
- Only ONE element should be visually unique per scene

## Serial Position Effect
- First (primacy) and last (recency) items are remembered best
- Hook scene sets first impression → must be strongest
- CTA scene is last → must be clearest

## Zeigarnik Effect
- Incomplete tasks are remembered better than complete ones
- Use visual tension (partial reveals, progress indicators) to drive engagement

## Law of Proximity
- Elements near each other are perceived as related
- Use deliberate spacing to group or separate information

## Law of Common Region
- Elements enclosed together are perceived as a group
- Use cards, containers, and borders to create visual groups

## The Golden Ratio (φ = 1.618)
- Apply to type scale: each step = previous × 1.618
- Apply to spacing: margin ratios follow golden ratio
- Apply to layout: divide sections by golden ratio
