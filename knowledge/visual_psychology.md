# Visual Psychology — Designer Knowledge

## Gestalt Principles for Motion Graphics
| Principle | Rule | Application |
|-----------|------|-------------|
| **Proximity** | Elements near each other = grouped | Group related stats, keep unrelated items far |
| **Similarity** | Same style = same category | Consistent colors for same data type |
| **Closure** | Brain fills gaps | Partial circles/shapes create intrigue |
| **Figure/Ground** | Separate foreground from background | Use depth layers, blur, opacity |
| **Continuity** | Eye follows smooth paths | Align elements in curves or lines |
| **Common Fate** | Elements moving together = grouped | Animate related items simultaneously |

## Visual Hierarchy (Priority Order)
1. **Size** — Biggest gets seen first (3× rule for focal elements)
2. **Contrast** — Highest contrast gets attention
3. **Color** — Saturated vs desaturated
4. **Position** — Top-left for LTR, center for titles
5. **Isolation** — White space around = importance
6. **Weight** — Bold vs light typeface

## The Squint Test
If you squint at your design and can't immediately identify:
- ONE dominant element
- A clear visual path
- Figure vs ground separation
→ The design fails. Redesign.

## Attention Span Rules
- **2 seconds**: User decides to stay or leave
- **7±2 items**: Maximum cognitive load (Miller's Law — we use ≤4 per Hick's Law)
- **F-pattern**: How users scan text-heavy layouts
- **Z-pattern**: How users scan image-heavy layouts

## Shape Psychology
| Shape | Association | Use For |
|-------|------------|---------|
| **Circles** | Trust, community, infinity | Avatars, badges, soft CTAs |
| **Squares/Rectangles** | Stability, order, structure | Cards, containers, grids |
| **Triangles** | Energy, direction, danger | Arrows, warnings, dynamic content |
| **Organic curves** | Natural, friendly, creative | Backgrounds, decorative elements |
| **Sharp angles** | Edgy, modern, aggressive | Tech, sports, breaking content |
